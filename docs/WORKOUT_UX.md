# Workout UX (Phase 1.5)

## Manual Flow

1. Start Workout
2. Select Exercise
3. Enter weight/reps
4. Save Set
5. Rest timer starts
6. Repeat
7. Finish Workout

## NFC Flow

1. Tap exercise tag
2. Exercise becomes active
3. Enter weight/reps
4. Save Set
5. Rest timer starts

## QR Flow

1. Scan QR code at machine
2. Exercise becomes active
3. Enter weight/reps
4. Save Set
5. Rest timer starts

## Notes

- Current implementation is helper-based and YAML-only.
- Current behavior is single-user oriented but keeps entity naming and structure compatible with future multi-user extension.
